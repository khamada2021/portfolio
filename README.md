Mutahar Mohammed — Portfolio

Overview

This is a modern, minimal static portfolio built with HTML, CSS, and vanilla JavaScript. It contains sections for hero, about, skills, projects, blog, education, and contact.

How to customize

- To change the profile photo, replace `assets/profile.jpg` with your image (keep the filename).
- Update project repository links in `index.html` under the projects section (replace `#` with your repo URLs).
- Edit colors and typography in `css/style.css` using CSS variables at the top of the file.

Local preview

Open `index.html` in a browser. For a quick local server, use Python:

```bash
python -m http.server 8000
# then open http://localhost:8000
```

Accessibility notes

- A skip link has been added for keyboard users.
- The mobile navigation is keyboard accessible and can be closed with the Escape key.
